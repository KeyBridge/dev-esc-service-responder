Example SAS / ESC status listener

The REST resources are described in the application.wadl file.

Ping listener

  • PUT /dev/esc/rs/listen/ping

DPAC status listener

  • PUT /dev/esc/rs/listen/dpac

The context root for this application is "/dev/esc/rs". This is configured in the web.xml file.

Web resources are exposed under the "listen" application path. This is configured in the ApplicationConfig.java class.
